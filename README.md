# Khushi Mundra — Chai pe Charcha

Personal portfolio site, built from the "3b — Chai pe Charcha" design direction.

## Develop

```
npm install
npm run dev
```

## Build

```
npm run build
```

Outputs static files to `dist/` — deploy that folder anywhere that serves static files (Vercel, Netlify, GitHub Pages, etc).

## Structure

- `src/components/` — one component per page section (Nav, Hero, Chat, Work, Writing, Wandering, Footer)
- `src/assets/photos/` — real photos, extracted from the Claude Design prototype's image slots
- `public/khushi-mundra-resume.pdf` — linked from the "grab my resume" button
